Puzzle Arcadia

Choose a variety of games here!

Hangman

A word-guessing game where players try to guess a hidden word by suggesting letters within a limited number of guesses while avoiding a stick figure being drawn.

2048

A tile-matching puzzle game where players combine numbered tiles by sliding them across a grid to create a tile with the number 2048.

Sudoku

A number-placement puzzle where players fill a 9x9 grid with digits so that each column, row, and 3x3 subgrid contains all digits from 1 to 9 without repetition.
